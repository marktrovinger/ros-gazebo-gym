Turtlebot3 Environment
======================

Turtlebot3_ is the new version of the classic Turtlebot3 created by
`ROBOTIS <http://www.robotis.us/>`__. It has configurable systems.

.. _Turtlebot3: http://www.robotis.us/turtlebot-3/
